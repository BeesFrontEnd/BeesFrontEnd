**文章目录**

> DOM（Document Object Module）文档对象模型，W3C组织推荐的处理可扩展标记语言的标准编程接口，可以通过JavaScript操作页面的元素，包括
添加、移动、改变和移除等操作。

W3C DOM标准

- 核心DOM：针对任何结构化文档的标准模型。
- XML DOM：针对XML文档的标准模型。
- HTML DOM： 针对HTML文档的标准模型。

DOM节点

HTML文档中的所有内容都是一个节点，整体的结构与数据结构中的树相同。
